오늘은 블로그를 완성한 기념으로, 3개월동안 jekyll과 씨름한 기념으로, window 에서 jekyll 설치하는 방법을 매우 상세하게 알려주려고 합니다.<br><del>window 개객기</del>

우선 오로지 <b class="blue">CMD</b> 로만 합니다.

<b class="blue">CMD</b>를 <b class="red">관리자 권한</b>으로 실행시켜주세요.<br><del>설마 CMD를 모르진 않겠지..</del>

그 다음 윈도우 패키지 관리자인 <b class="blue">Chocolatey</b>를 설치할겁니다.

{% highlight text %}
@powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
{% endhighlight %}

요걸 복사하신 후에 <b class="blue">CMD</b> 에서 "마우스 오른쪽 클릭 – 붙여 넣기" 를 하신 후 실행하면 설치가 됩니다.

설치가 완료되었다면 이제 <b class="blue">ruby</b>와 <b class="blue">jekyll</b>을 설치해줄 차례입니다.

&nbsp;

<b class="blue">CMD</b> 에서 <code>choco install ruby -y</code> 를 입력해 <b class="blue">ruby</b>을 설치합니다.

<code>/* 2016-11-13 추가 / 프론트 엔드 개발 그룹 - 강동훈 님 */</code><br>
Windows 7 64bit에서 설치 중입니다. 루비에서 막혀서<br>"Unable to download data from https:rubygemsorg - SSL_connect returned=1 errno=0 state=SSLv3 read server certificate B: certificate verify failed"

이 에러 뜨시는 분은 아래 게시물에서 "For windows users" 부분 참고해서 ruby 설치 후 jekyll 설치 진행하시면 됩니다!<br><a href="http://stackoverflow.com/a/27641786/7152877" target="_blank">http://stackoverflow.com/a/27641786/7152877</a><br>
<code>/* end */</code>

&nbsp;

그리고 <code>choco install ruby2.devkit</code>를 입력해 <b class="blue">devkit</b>도 깔아줍니다

그 다음 설정인데요 <b class="blue">CMD</b>를 재실행한 다음 <code>cd C:\tools\DevKit2</code>를 입력한 뒤 <code>ruby dk.rb init</code>를 입력해 활성화한 다음.<br>
<b>C:/tools/DebKit2/config.yml</b> 파일을 열어 하단에 <code>- C:/tools/ruby23</code>을 입력하고 저장해줍니다.

다시 <b class="blue">CMD</b>로 돌아와서 <code>ruby dk.rb install</code> 을 쳐주고 업데이트 표시가 나오면 성공입니다.

&nbsp;

<b class="blue">CMD</b>를 재실행한 뒤  <code>gem install jekyll -v '3.2.1'</code> 를 입력해 <b class="blue">jekyll</b>을 설치해줍니다.

에러가 난다면 <code>gem source -a http://rubygems.org</code> 를 써줘서 <b>rubygems.org</b> 를 인식하게 한 후 재시도 합니다.

&nbsp;

자 드디어 <b class="blue">jekyll</b> 이 설치되었지만 <code>jekyll new blog</code> 를 하는 순간 에러가 나며 파일이 덜 생성될 겁니다.

다시 <b class="blue">CMD</b> 창에서 <code>gem install bundler</code>를 입력해 <b class="blue">bundler</b> 를 설치해줍니다.<br>중간에 질문이 나온다면 다 Y 해줍니다.

그리고 <code>gem install minima -v '2.0.0'</code>을 입력해 <b class="blue">minima</b>도 설치해줍니다.

그리고 <code>gem install jekyll-feed -v '0.8.0'</code>을 입력해 설치해줍니다.

그리고

<code>choco install libxml2 -Source "https://www.nuget.org/api/v2/"</code>

<code>choco install libxslt -Source "https://www.nuget.org/api/v2/"</code>

<code>choco install libiconv -Source "https://www.nuget.org/api/v2/"</code>
